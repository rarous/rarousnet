    <h1>Naše první stránka.</h1>

prohlídnout výslednou stránku

Význam řádků

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN">
tento řádek říká, že se jedná o XHTML dokument. Prohlížeč díky němu ví, jak má se souborem zacházet.
<html lang="cs">
zde začíná samotný XHTML dokument, atribut lang říká, že je dokument v českém jazyce
<head>
zde začíná hlavička dokumentu, ta se většinou nezobrazuje v prohlížeči, ale je důležitá pro správné zobrazení stránky
<title>První stránka</title>
titulek stránky, zobrazuje se v horním proužku prohlížeče
</head>
ukončuje hlavičku
