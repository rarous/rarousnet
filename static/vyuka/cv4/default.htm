Dnes se budeme učit vytvářet tabulky. Tabulka slouží k uspořádání tabulkových dat do přehledné formy. Ještě před pár lety se tabulky používaly k definování rozložení prvků na stránce. Tato technika je již ale překovaná mocnějším nástrojem — kaskádní styly (CSS).

Takto může vypadat krátký výpis z databáze kontaktů. K takovému výpisu je asi tabulka nejvhodnější pro svoji přehlednost. Podíváte-li se do jejího kódu, pravděpodobně se v něm někde ztratíte, přesto že jsem zvolil metodu zápisu, která je pro zpracování člověkem příjemnější. Nyní tedy k vysvětlení použitých elementů. Tabulka je tvořena elementem table, ten může mýt jednu hlavičku thead a patičku tfoot. Dále pak má tělo tabulky tbody. Je to tedy podobné struktuře samotného HTML souboru. My jsme v naší ukázce žádnou patičku nepoužili.

Tagy pro práci s tabulkami

table
Úvodní tag tabulky. Má několik atributů, které definují vzhled tabulky, ale ty se v dnešní době nahrazují pomocí CSS. Dále má atribut summary, kde se píše shrnutí obsahu tabulky — informační výtah. Tento atribut se používá pro zpřístupnění obsahu např. zrakově postižených lidem, kteří nevidí jak tabulka vypadá a čtečka, která jim čte obsah tabulky nerozeznává jednotlivé sloupce a řádky.
caption
Obsahuje titulek tabulky — popiska, nadpis, je lepší jej uvádět pro lepší orientaci v dokumentu.
tbody
Obsahuje vlastní data tabulky. Měl by se vyskytovat ve všech tabulkách.
thead
Obsahuje záhlaví tabulky, většinou názvy sloupců.
tfoot
Obsahuje patičku tabulky, např. nějaké výsledné statistické hodnoty

Správně by se u dlouhých tabulek mělo zobrazovat thead a tfoot na každé stránce (pokud jde o výtisk) ap. Většinou tomu ale tak není.

tr
Definuje řádek tabulky. Je to jediný element, který může být potomkem tbody, thead a tfoot.
td
Definuje jednu buňku tabulky. Může být pouze jako potomkem tr.
th
Definuje hlavičkovou buňku tabulky (první řádek v dnešní ukázce). Může být pouze jako potomkem tr.

Kaskádní styly (CSS)

Kaskádní styly jsou jedním z nejmodernějších prostředků, jak přizpůsobit vzhled webových dokumentů a to nejen HTML a XHTML, ale i jiných XML souborů. V tomto dokumentu je použito několik stylových předpisů pro definici vzhledu tabulky a změnu vzhledu poznámky oproti normálním odstavcům. Jednotlivé definice se píší v následujícím tvaru:

selektor {
    pravidla;
    ...
}

Selektorů máme několik druhů:

  1. přímé: tag — jsou to názvy tagů, vybírají všechny elementy s daným jménem a na ně pravidla aplikují.
  2. class selektory: .hodnota_atrtibutu_class — vybírají elementy, které mají atribut class shodný s textem za tečkou
  3. id selektory: #hodnota_atributu_id — vybírají element, který má atribut id shodný s textem za mřížkou

Dále se dají selektory kombinovat a pomocí různých operátorů dále přibližovat, ale to už je složitější a necháme si to na další cvičení. Tady máme ukázku stylů aplikovaných na tuto stránku.

table {
    border-collapse: collapse; /* zruší mezeru mezi buňkami */
    width: 75%; /* nastaví šířku tabulky na 3/4 šířky stránky */
}

td, th {
    border: 1px solid #AAA; /* nastaví tenký rámeček šedé barvy pro buňku tabulky */
    padding: 2px 4px; /* lehké odsazení obsahu od rámečku */
}

.poznamka {
    background-color: #CCC; /* pozadí stříbrné barvy */
    font-size: small; /* malé písmo */
}
